## 🔧 Configuration

1. **Set up API Key:**
   - Add your Gemini API key to `.env` file
   - Key name: `VITE_GEMINI_API_KEY`

2. **Environment File:**
   ```env
   VITE_GEMINI_API_KEY=your_key_here
   ```

3. **Security:**
   - `.env` is in `.gitignore` - never commit it
   - Use `.env.example` as reference template
