# 🧠 AI Quiz Master

An interactive quiz application powered by Google Gemini API. Users can log in, select topics, and take AI-generated MCQ quizzes with instant feedback and explanations.

## Features

✅ **User Authentication** - Local login/signup system
✅ **AI-Generated Questions** - Dynamic MCQ generation using Google Gemini API
✅ **Multiple Topics** - Choose from 10+ subjects
✅ **Interactive UI** - Beautiful gradient design with smooth animations
✅ **Instant Feedback** - Immediate results with AI-generated explanations
✅ **Score Tracking** - Real-time score display
✅ **Responsive Design** - Works on desktop and mobile

## Setup Instructions

### 1. Install Dependencies
```bash
npm install
```

### 2. Configure Environment Variables

Create a `.env` file in the root directory with your Gemini API key:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

The `.env.example` file shows the required format.

### 3. Get Your API Key

1. Go to [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Create a new API key for the Gemini API
3. Copy the key and paste it in your `.env` file
4. **Important:** Never commit `.env` to git - it's already in `.gitignore`

### 4. Run the Development Server
```bash
npm run dev
```

Visit `http://localhost:5173` in your browser.

### 5. Build for Production
```bash
npm run build
```

## How to Use

1. **Sign Up/Login** - Create an account or login with existing credentials (stored locally)
2. **Select Topic** - Choose from available topics (AI Basics, Python, Web Development, etc.)
3. **Set Questions** - Choose number of questions (1-10)
4. **Start Quiz** - Click "Start Quiz" to begin
5. **Answer Questions** - Select your answer and click "Submit Answer"
6. **View Feedback** - See if you're correct and read AI-generated explanations for wrong answers
7. **Track Score** - Monitor your score throughout the quiz
8. **End Quiz** - Complete or end early to return to the topic selection

## API Integration

The app uses the Gemini 1.5 Flash model for fast question generation:

```javascript
const API_URL = 'https://generativelanguage.googleapis.com/v1beta/models/gemini-1.5-flash:generateContent';
```

Questions are generated based on:
- Selected topic
- Number of questions requested
- Each question includes options and explanations

## Troubleshooting

**API Key Error:**
- Ensure `.env` file exists in the root directory
- Check that `VITE_GEMINI_API_KEY` is correctly formatted
- Verify API key is active in Google Cloud Console

**Questions Not Loading:**
- Check browser console for error messages
- Ensure internet connection is active
- Verify API quota hasn't been exceeded

**Styling Issues:**
- Clear browser cache
- Ensure CSS files are in the src directory
- Check that imports match file paths
